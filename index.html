Configurazione Firebase

Inserisci le credenziali del tuo progetto Firebase per abilitare i dati condivisi in tempo reale.

Come ottenere le credenziali:
1. Vai su console.firebase.google.com
2. Crea un nuovo progetto (gratuito)
3. Aggiungi un'app Web (icona </>)
4. Copia i valori da firebaseConfig
5. In Realtime Database → Crea database → Regole → imposta "read": true, "write": true
